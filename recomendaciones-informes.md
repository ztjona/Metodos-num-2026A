
# Secciones de un informe
* **_Abstract_**: resumen
* **Introducción**: relevantes, objetivo. (Encontrar un algoritmo, comprobar experimentalmente).
* **Marco teórico**: conceptos y teorías existentes. (Problema del viajero). 
* **Metodología**: descripción detallada de lo que se hizo. Información suficiente para replicar el experimento.
    - Programación en python, con estas herramientas. 
    - Enunciar ideas descartadas, o alternativas ignoradas. 
    - Implementamos estos dos algoritmos A, B.
    - Planteamiento de los experimentos.
    - Hardware utilizado. 

* **Resultados**: resumen de las pruebas, variantes y resultados obtenidos. Se recomienda usar tablas, gráficos, etc. para mostrar los resultados de manera clara y concisa.
    - Se probó algoritmo A con configuración 1a, 2a, 3a.
    - Se probó algoritmo B con configuración 1b, 2b, 3b.


* **Análisis**: interpretación de los resultados, _comparación con otros trabajos_. 
    - El algoritmo A obtuvo mejores resultados que el algoritmo B en la mayoría de las configuraciones.
    - Sin embargo, el algoritmo B mostró un mejor rendimiento en la configuración 2b, lo que sugiere que podría ser más adecuado para ciertos tipos de problemas.

* **Conclusiones**: resumen de los hallazgos, implicaciones y posibles direcciones futuras. 
    - En conclusión, saqué mala nota porque no estudié, copié los deberes, y el taller hice con ChatGPT. 

* **Recomendaciones**: sugerencias para el futuro. 
    - Se recomienda estudiar más para mejorar la nota en el próximo informe. 
    - Se recomienda hacer los deberes por uno mismo para aprender mejor. 
    - Se recomienda hacer el taller sin ayuda de ChatGPT para desarrollar habilidades de programación.

* **Referencias**: lista de fuentes utilizadas para el informe. Al menos 2.
